Project Overview

About This Project

Audio Speech to Sign Language Converter is an interactive web application designed to bridge communication gaps by transforming spoken language into visual sign language animations.

What It Does

The system captures spoken audio, converts it into text using speech recognition, processes the transcript using Natural Language Processing (NLP), and displays corresponding sign language animations based on detected keywords.

Key Features

  • Real-time Speech-to-Text Conversion
  • NLP-based Keyword Detection
  • Dynamic Sign Language Animation Playback
  • Web-based Interactive Interface

Purpose

This project demonstrates how speech recognition and natural language processing can be integrated into assistive technology systems to promote accessibility and inclusive communication.
